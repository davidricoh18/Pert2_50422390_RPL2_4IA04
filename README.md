Pada praktikum RPL 2 pertemuan 2, saya mempelajari anatomi dasar sebuah class dalam Java, mulai dari deklarasi atribut, method, hingga constructor. 
Selain itu dipelajari juga struktur umum aplikasi Java yang terdiri dari package, class, dan method utama. 
Materi ini dilanjutkan dengan konsep Object-Oriented Programming (OOP) seperti encapsulation, inheritance, dan polymorphism untuk membangun program yang modular, reusable, serta lebih mudah dipelihara.
